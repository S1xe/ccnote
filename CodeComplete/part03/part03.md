- 在程序开始时初始化工作内存

P245

计算变量跨度(span):对一个变量引用的集中程度

```
例1
a=0
b=0
c=0
a=b+c
//那么,a的跨度是2,b的跨度是1,c的跨度是0
```
```
例2
a=0
b=0
c=0
b=a+1
b=b/c
//b的跨度为(1+0)/2,第一次对b引用和第二次对b引用分别为1,0
```
